## Autopkg

A script to setup your own pacman repository. Fetches PKGBUILDs from AUR, [extensions.gnome.org](https://extensions.gnome.org), and git repositores automatically.

### Dependencies

* Python 3.6.4+
* `coreutils`
* `pacman`
* `devtools`
* `btrfs-progs`
* `sudo`
* `gnupg`
* `git`
* `bash`
